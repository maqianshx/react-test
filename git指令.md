先建立远程仓库,在github或gitlab或gitee
在本地去init项目
第一步 创建自己的项目  例如:create-react-app
第二步 git init 初始化git
后面就是开发的流程了

### git status
查看当前工作区修改的文件的状态
### git add xxx
把当前修改的文件,提交到暂存区
### git commit
把暂存区的文件提交到本地仓库
### git log
查看当前分支的commit信息
### git branch
查看分支信息
### git checkout
切换分支
### git push
提交到远程
### git pull 
拉取远程代码
### git clone 
克隆远程仓库
